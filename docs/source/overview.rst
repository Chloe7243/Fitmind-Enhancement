Overview
========

About FITMIND
-------------

**FITMIND: Exercise & Stress Tracker** is an all-in-one wellness web application tailored to university students seeking to take control of their mental and physical health. The application allows users to log daily stress levels, track exercise routines, record personal reflections, set custom reminders, and access a guided breathing tool — all within a unified, responsive interface.

Developed to promote consistency, simplicity, and meaningful feedback, FITMIND empowers users to form healthier habits through small, sustainable actions. By linking stress management with physical activity, the app delivers a balanced approach to student well-being.

FITMIND supports:

- Stress logging with personalized recommendations
- Exercise tracking with real-time progress visualization
- Guided breathing sessions for quick relief
- Journaling features for self-reflection
- Reminders to reinforce routine and consistency
- Secure user authentication to protect personal data

Project Purpose
---------------

The increasing demands of academic life often leave students with little time or energy to prioritize self-care. FITMIND was created to address this problem by offering a lightweight, intuitive platform where students can check in with themselves — emotionally and physically — without pressure or complexity.

The app serves both as a **wellness tracker** and a **motivational tool**, helping users identify patterns in their stress and activity levels and adjust their routines accordingly.

Key Features
------------

FITMIND is built around eight core pages, each designed for ease of use and targeted functionality:

- **Landing Page** – Introduces the app’s purpose and invites users to explore or sign up.
- **Register & Login Pages** – Secure account creation and session-based access to personalized data.
- **Homepage (Dashboard)** – Acts as the central hub, giving users a summary and quick access to main features.
- **Stress Page** – Lets users log stress levels on a 1–5 scale using mood-based emojis, with space for notes and automatic personalized feedback.
- **Exercise Page** – Allows users to record physical activities, view session-based live progress charts, and receive smart suggestions based on stress data.
- **Breathe Flow Page** – A publicly accessible, guided breathing tool that helps users reduce stress through simple, timed inhale–hold–exhale cycles.
- **Notes Page** – A free-form journaling space where users can write reflections and search past entries using a built-in filter.
- **Reminder Page** – Enables users to set custom reminders with time selectors and pop-up alerts with sound, working across all pages of the app.

Target Audience
---------------

The primary audience for FITMIND is university students — individuals who often experience high stress levels due to tight academic deadlines, lifestyle adjustments, and limited time for self-care. However, the application is versatile and can be useful to anyone interested in developing mindful wellness habits in a flexible, data-driven way.

Design Philosophy
-----------------

FITMIND was designed with a focus on five key values:

- **Simplicity** – Every interaction is fast, intuitive, and requires minimal explanation.
- **Personalization** – Recommendations are responsive to the user's input and needs.
- **Accessibility** – The app is fully responsive and works across mobile, tablet, and desktop devices.
- **Security & Privacy** – All personal data is stored securely and only accessible through authenticated sessions.
- **Empathy over gamification** – The app avoids pressure-inducing features like badges or streaks, focusing instead on encouragement and usability.

Technology Stack
----------------

FITMIND was developed using the following technologies:

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Python (Flask)
- **Database**: SQLite
- **Version Control**: Git and GitHub
- **Documentation**: Sphinx with Read the Docs integration

What Makes FITMIND Different?
-----------------------------

While many wellness apps focus exclusively on fitness or mental health, FITMIND bridges both domains. It uniquely connects how users feel with how they move, using that data to offer **context-aware, actionable suggestions**.

The Breathe Flow tool, for instance, is available without logging in — allowing any user, regardless of account status, to experience immediate stress relief. This commitment to **inclusive well-being** sets FITMIND apart.

Whether you're logging an exercise, taking a moment to breathe, or reflecting in a journal, FITMIND is designed to meet you where you are — and help you take the next small step forward.
